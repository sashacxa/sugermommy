# Markdown基本语法

### 1. 标点切换

用半角标点，即英文标点，如`,.[]()/>:|""`，加上空格。

### 2. 标题

```
# title 1
## title 2
### title 3
...
###### title 6
```

### 3. 各种字体

```
**强调（加粗）**
*斜体*
***斜体和加粗***
==高亮== (vsc中)
~~删除~~
<u>下划线</u>
> 引用
> > zaiyinyong
```

- 效果展示：

  **强调**

  *斜体*

  ***斜体和加粗***

  ~~删除~~

  <u>下划线</u>

  > 引用
  >
  > > 

  > > zaiyinyong

### 5. 注释

```
> How to pass IELTS[^1]?
[^1]: An English Examination. 
```

* 效果展示：

  > How to pass IELTS[^1]?

  [^1]: An English Examination.
